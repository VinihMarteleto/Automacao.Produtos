# Automacao.Produtos

Projeto simples para automação de produtos.

Conteúdo:
- `codigo.py`
- `auxiliar.py`
- `produtos.csv`

Como publicar no GitHub:

1. Instale o GitHub CLI: `gh auth login`
2. Execute (na pasta do projeto):

```
gh repo create Automacao.Produtos --public --source=. --push
```

Ou crie o repositório via web e rode:

```
git remote add origin https://github.com/SEU_USUARIO/Automacao.Produtos.git
git branch -M main
git push -u origin main
```
